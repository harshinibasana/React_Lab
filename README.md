# OCU CSCI 6333 - Assignment 04
This is a ready-to-run React project implementing the assignment:
- React Router v6 routes: /, /about, /contact, /feedback, /account/profile, /account/settings, /thank-you
- Controlled Contact form with validation and programmatic navigation
- Uncontrolled Feedback form using useRef
- Nested Account routes with <Outlet />

To run:
1. `npm install`
2. `npm start`

Note: this project expects create-react-app environment (react-scripts). If you use Vite, adjust scripts and index mounting accordingly.
